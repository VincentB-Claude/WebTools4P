Invoice Tool

Công cụ đối soát hóa đơn PDF với GRPO từ SAP.
Yêu cầu đăng nhập bằng tài khoản @pizza4ps.com.

Tải về (Windows .exe)

Cập nhật tự động · Không cần cài đặt

Hướng dẫn sử dụng

1

Tải về & chạy

Download file .zip, giải nén rồi double-click file .exe để mở — không cần cài đặt thêm.

2

Tab 1 — Convert PDF

Chọn thư mục chứa hóa đơn PDF, nhấn Run để trích xuất tự động thành file Markdown.

3

Tab 2 — Đối soát GRPO

Chọn thư mục MD vừa tạo và file GRPO từ SAP, nhấn Run để so khớp tự động.

4

Xuất kết quả

Nhấn Export Excel để lấy báo cáo, nhấn Export SAP để tạo file upload SAP.
